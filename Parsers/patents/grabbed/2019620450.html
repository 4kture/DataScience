Название базы данных:
Конструкции технологических узлов верхней одежды из натурального коротковолосого меха

Реферат:
База данных (БД) предназначена для сбора, хранения, поиска, корректировки и обновления конструкций технологических узлов (КТУ) верхней одежды из натурального коротковолосого меха (КТУВОНКМ) плечевой группы пальтово-костюмного ассортимента. Формирование данного сегмента знаний является для дизайнеров и инженеров не самоцелью, а средством реализации творческой мысли при разработке и выполнении проектных работ по изготовлению швейных изделий современной моды. БД КТУВОНКМ позволяет на основе базовых знаний предметной области «Технология обработки изделий из натурального меха» изучать и выбирать конструкции технологических узлов верхней одежды из натурального коротковолосого меха пальтового-костюмного ассортимента плечевой группы в рамках образовательной и производственной деятельности. Актуальность разработки заключается в том, что она позволяет свести к минимуму многообразие технологических решений, структурировать их в соответствии с логикой последовательности изготовления верхней одежды из натурального коротковолосого меха и помогает специалисту - технологу выбирать наиболее оптимальные варианты конструкций технологических узлов, формировать требуемую нормативно-технологическую документацию, быть в курсе новых событий в области индустрии моды. БД КТУВОНКМ плечевой группы обладает декларативными знаниями, имеет файловую систему, совместимую со всеми операционными системами, где размещена база данных. Разработка выполнена в виде дерева каталогов, которые содержат графическую и текстовую информацию. Архитектура электронной базы данных имеет вид «клиент-сервер». Для создания связей между данными используются РНР модули (скрипты), преобразующие и составляющие необходимую последовательность по динамически создаваемой структуре. Для БД КТУВОНКМ плечевой группы выделено четыре основных вида данных: графические - в виде jpg документов, табличные - в виде HTML файлов с таблицами, структурные - в виде текстовых файлов с уникальными именами и форматированным содержанием, архивные - в виде DWG (ACAD) файлов. Манипулирование данными упрощается за счет опционального интерфейса, гибкой системы поиска и введенной кодировки позиций и пунктов БД. При добавлении данных в БД КТУВОНКМ плечевой группы используется технология РНР, которая обеспечивает добавление, изменение и просмотр данных.

Вид и версия системы управления базой данных: SQL Server, HTML
