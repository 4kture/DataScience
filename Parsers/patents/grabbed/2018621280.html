Название базы данных:
База данных сопутствующих изображений морского льда разной сплоченности радиолокатора Sentinel-1, измерений микроволнового радиометра AMSR2 и значений сплоченности морского льда, восстановленных по данным AMSR2, а также данных по скорости приводного ветра, восстановленной по данным AMSR2

Реферат:
База данных представляет собой совокупность отобранных файлов спутниковых данных, содержащих обработанные активные и пассивные измерения над морским льдом разной сплоченности, выполненные в разных диапазонах электромагнитного спектра. База данных состоит из объединенных файлов 4-х типов данных: радиолокационных изображений Sentinel-1, радиометрических измерений AMSR2 на частоте 89 ГГц разрешения 6.25х6.25 км, данных по сплоченности морского льда на основе измерений AMSR2 и данных по скорости приводного ветра Remote Sensing Systems, отобранных на основании визуального анализа с целью включения в базу всего диапазона изменчивости сплоченности морского льда – от 0 до 100 %. Временной охват – 2015 – 2016 гг., географический охват – моря Арктики. Радиометрические данные представляют собой горизонтально и вертикально поляризованные радиояркостные температуры микроволнового излучения на частоте 89 ГГц для региона Арктики, средние для дневных и ночных измерений. Осреднение и гридирование данных AMSR2 проведено для измерений А-сканов, что соответствует разрешению 6.25х6.25 км.

Вид и версия системы управления базой данных: Excel; программа, работающая в среде MATLAB
