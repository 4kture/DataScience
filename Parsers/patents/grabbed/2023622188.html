Название базы данных:
СМИ Тулы и Тульской области XIX–XXI вв.

Реферат:
База данных содержит сведения о СМИ и медиахолдингах города Тулы и Тульской области с 7 января 1838 года по настоящее время. Информация в базе представляет собой множество документов, полученное в результате горизонтальной декомпозиции типологических признаков СМИ по шкале времени. Каждый документ содержит сведения в виде значений, присвоенных следующим ключам: «название»; «учредитель»; «датаот» (дата выпуска первого номера); «датадо» (дата выпуска последнего номера); «канал», «архивная запись» (пояснение названия каждого СМИ или тип СМИ); «аннотация» (имеет смысл описания в каталожной библиотечной карточке); «редакция» (состав редакции, вложенный документ с ключами: «журналисты», «фотографы», «кинооператоры», «телеоператоры», «художники» и т. п.); «основные рубрики»; «статьи»; «периодичность»; «примечания» (исторические заметки связанные с деятельностью редакции и ее сотрудников); «электронная версия»; «издатель»; «дизайнер»; «тираж»; «формат»; «число полос»; «оформление» (тип бумаги, красочность и т. п.); «типография»; «место печати»; «шеф-редактор»; «генеральный директор»; «автор идеи»; «состав» (состав медиахолдингов); «location» (географические координаты редакции, дирекции холдинга и т. п.); «форма распространения»; «эфир» (сетка вещания); «цена». Совокупность значений ключей: «название», «датаот», «датадо», «канал», «глава редакции» и «учредитель» уникальна для каждого документа — так некоторые СМИ декомпозированы в несколько документов, что отражено единым для них значением специального ключа «PCID». База данных предназначена для построения типологий СМИ, изучения истории, организации онлайн-ресурсов, приложений для задач регионоведения и представлена в формате JSON. Тип ЭВМ: PC, Server. ОС: Мультиплатформенная.

Вид и версия системы управления базой данных: NoSQL, MongoDB
