Название базы данных:
База данных матриц рассеяния света для атмосферных несферических аэрозольных частиц размерами от 5 до 100

Реферат:
База данных предназначена для интерпретации данных лазерного зондирования атмосферного аэрозоля, что необходимо для исследования возможности восстановления микрофизических характеристик аэрозольных образований из данных лазерного зондирования для задач моделирования климата, мониторинга и прогнозирования окружающей среды. База данных представляет собой решение прямой задачи рассеяния света на аэрозольных частицах и позволяет получать информацию о микрофизических характеристиках по данным лазерного зондирования атмосферы. База данных содержит совокупность матриц обратного рассеяния света. Тип ЭВМ: Архитектура х86, х64; ОС: Windows, Linux.

Вид и версия системы управления базой данных: PostgreSQL
